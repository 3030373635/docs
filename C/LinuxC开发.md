## 一、printf

```c
printf() -- 讲变量的内容输出到显示器上

    四种用法
        1. printf("字符串")
        2. printf("输出控制符",输出参数)
        3. printf("输出控制符1 输出控制符2",输出参数1,输出参数2)
        4. printf("输出控制符 非输出控制符",输出参数)


    输出控制符:
        指定一种输出格式，将内存中的二进制数据打印出来
        %d        --    int
        %ld        -- long int
        %c         --char
        %f         -- float
        %lf     -- double
    %x/%X/%#X     -- int或者log int 或者 short int
        %x         --同上
        %s         --字符串

    为什么需要输出控制符
        1. 二进制代码可以表示数据，也可以表示其他东西
        2. 如果二进制代码表示数据的话，那么同样的二进制代码以不同的输出格式输出的话，就可以得到不同的结果，比如
            int a = 15
            printf("16进制a=%#X",a)
```

## 二、scanf

> &取地址符，把输入的ab的值放在取到的内存地址中

```c
用法一
    scanf("输入控制符",输入参数)
用法二
    scanf("非输入控制符 输入控制符",输入参数)
    例如：
        scanf("m%d",i)
        //此时输入的时候必须以m开头后面才是数字
```

## 三、scanf与printf总结

```c
printf:
    非输出控制符，在printf里面原样输出

scanf:
		1. 输入的数据中间可以随意空格
  
    2. scanf中尽量不要使用非输入控制符，尤其是\n
  
    3. 非输出控制符，在scanf里面原样输入

    4. 应该编写代码对用户的非法输入,做适当处理
        例子:
            scanf("%d",&i);
            printf("i = %d\n",i);


            scanf("%d",&j);
            printf("j = %d\n",j);

            上面这个例子，如果用户输入123m，123m被存入缓冲区中，
            123被第一个scanf从缓存区取走了，留下了m，第二个scanf就会去缓冲区继续取，取到了m，给了j，但是j是int类型，所以会报错

        如何适当处理:
            在第二个scanf准备去缓冲区取东西的时候，清空缓存区，那么用户就可以再次输入
			while(getchar() != '\n');
